How I installed Go on my corporate Windows laptop without administrator rights

10 August 2016

I have started by going to https://golang.org/ ;-)

There, I discovered the A Tour of Go which allows you to start learning the Go language without installing anything. Brilliant!

Remark: The A Tour of Go is built on top of Go Playground, a web service that runs on golang.org’s servers. The service receives a Go program, compiles, links, and runs the program inside a sandbox, then returns the output. You can also use this service to share your single source file Go program in a demo-able mode; try this!
